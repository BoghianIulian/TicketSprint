README – TicketSprint


TicketSprint este o aplicație web dezvoltată din pasiunea pentru sport și tehnologie, cu scopul de a moderniza procesul de rezervare a biletelor la evenimente sportive. Ideea proiectului s-a născut din dorința de a crea o platformă care oferă o experiență personalizată fanilor sportului, reunind funcționalități esențiale într-un mod intuitiv și eficient.

Funcționalități pentru utilizatori

- Înregistrare și autentificare securizată cu token JWT

- Vizualizare și filtrare a evenimentelor după criterii precum sport, oraș, locație

- Sistem de favorite pentru echipe sau sportivi

- Recomandări personalizate în funcție de preferințe

- Selectare interactivă a locurilor în sală/stadion

- Blocarea temporară a locurilor selectate (15 minute)

- Coș de cumpărături și posibilitate de achiziție fără cont

- Generarea biletelor în format PDF, cu cod QR

- Trimiterea automată a biletelor pe email

- Secțiune personală cu biletele achiziționate și opțiune de anulare


Funcționalități pentru administratori

- Panou de administrare cu acces la gestionarea locațiilor, sectoarelor, subsectoarelor, participanților și evenimentelor

- Setarea prețurilor și alocarea zonelor de bilete

- Vizualizare interactivă a ocupării locurilor

- Editare dinamică a datelor în timp real prin JavaScript asincron

- Protecție logică asupra entităților legate de evenimente active


Tehnologii utilizate

- Frontend: HTML, CSS, JavaScript, Bootstrap, Razor Pages

- Backend: C#, ASP.NET Core

- Bază de date: SQL Server, Entity Framework Core

- Securitate: BCrypt, autentificare JWT

- Testare și dezvoltare: SwaggerUI, JetBrains Rider

Alte librării: MailKit pentru emailuri, PdfSharpCore pentru PDF-uri
